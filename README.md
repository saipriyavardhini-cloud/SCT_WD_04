# Advanced Task Manager

A modern Task Management Web Application built using HTML, CSS, and JavaScript.

## Project Description

The Advanced Task Manager helps users organize daily tasks efficiently. Users can create tasks, assign priorities, set due dates and times, track completion status, search tasks, filter tasks, and manage their workflow through a simple and responsive interface.

The application stores all task data using Local Storage, ensuring tasks remain available even after refreshing or reopening the browser.

---

## Features

### Task Management
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Undo completed tasks

### Task Information
- Task title
- Task description
- Due date
- Due time
- Priority levels (High, Medium, Low)

### Organization Tools
- Search tasks
- Filter tasks
  - All Tasks
  - Pending Tasks
  - Completed Tasks
- Sort tasks by priority

### Productivity Features
- Statistics dashboard
  - Total Tasks
  - Completed Tasks
  - Pending Tasks
- Overdue task detection
- Local Storage support
- Responsive design
- Dark Mode

## How to Run the Project

### Method 1: Open Directly

1. Download the project.
2. Extract the folder.
3. Open `index.html` in any modern web browser.

### Method 2: VS Code Live Server

1. Open the project folder in VS Code.
2. Install the "Live Server" extension.
3. Right-click `index.html`.
4. Click **Open with Live Server**.
